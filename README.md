# Organic Waste Detection using YOLOv9 (Demo)

This repository contains a **proof-of-concept computer vision project**
focused on detecting **organic waste** using object detection.

## Overview
- Model: YOLOv9
- Annotation Tool: CVAT
- Task: Object Detection (bounding boxes)
- Scope: Demo / Proof of Concept

The project is developed on a **small dataset** and is intended to
demonstrate how AI can be applied to waste management scenarios.
It is not a production-ready system.

## Motivation
Organic waste represents a significant portion of municipal waste.
Automated detection can support better waste segregation, composting,
and sustainable waste management practices.

## Demo
Sample detection outputs are available in the `sample_results/` folder.
A demo video is shared on LinkedIn.
